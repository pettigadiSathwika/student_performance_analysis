# Student Performance Analysis

This project analyzes student marks using Microsoft Excel.

Tools Used
- Microsoft Excel

 What I Did
- Created a dataset of student marks
- Calculated average marks for each student
- Identified the topper based on highest average
- Analyzed subject-wise performance
- Created charts for better visualization

 Conclusion
- Topper is Rama
- English subject has the highest scores
- Maths subject needs improvement
